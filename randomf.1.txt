NAME
       randomf - Random file generator

SYNOPSIS
       randomf [OPTION] DEST

DESCRIPTION
       Create a file of arbitrary size with random data. By default, the file is 1MB in size. To specify the size, see --size optional argument.

       -h, --help                
              Print this menu. Will not create any file if specified.
       -s, --size                
              Size of output file. (E.g., 512 represents 512 bytes.) Default value is 1MB. Optional units include B (bytes, default), KB, MB, GB.
